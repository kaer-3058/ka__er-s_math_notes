# ka__er-s_math_notes

卡儿的数学笔记的配套示例代码

This is the sample code for ka__er's math notes.

---

#### 文件说明

`ope_matrix_qr_iteration.py`：一元N次方程的数值方法的通解里，完整的基本QR迭代。

`ope_r_matrix.py`：一元N次方程的数值方法的通解里，计算R矩阵。

`zetazero-s_experimental_data.csv`：一种手算ζ零点的方法里，数值实验报告表格。近似公式的误差与速度分析表，在GitHub网页上点开就能预览。文件里，从左往右并排有四张表：

```
1.估计零点与真实零点的虚部比较
2.虚部误差较大的零点
3.各个零点的计算耗时
4.前N个零点的计算总耗时
```

`zetazero.py`：一种手算ζ零点的方法里，用Python计算黎曼零点的示例代码。

`zetazero_of_mathematics.txt`：一种手算ζ零点的方法里，在Mathematics计算黎曼零点的示例代码。

`zetazero_t_function_of_ggb.txt`：一种手算ζ零点的方法里，数值报告的实验数据-GeoGebra。

`zetazero_test_of_mma.txt`：一种手算ζ零点的方法里，数值报告的实验数据-Mathematics。
